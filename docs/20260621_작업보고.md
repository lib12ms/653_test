## 1. 작업 배경

기존 프롬프트는 분야별로 자유롭게 키워드를 추출하도록 설계되어 있었으나, AI가 다음과 같은 문제 키워드를 반복적으로 생성하는 현상이 확인됨:

- **줄거리 파생 복합어**: `자아희생`, `생존본능`, `지구생존` 등
- **상위분류어 단독 출력**: `사회과학`, `사회학`, `교육학` 등
- **감상어·메타표현**: `따뜻한`, `감정서사`, `서사구조` 등

→ **해결 방향**: 분야별로 키워드 유형을 사전에 지정하고, 그 유형 안에서만 추출하도록 프롬프트를 보수적으로 재설계

## 2. 프롬프트 구조 개편 (유형A-E 방식)

### 개편 전 (자유 추출 방식)
```
이 책은 사회과학 도서입니다.
- 사회현상·제도·이론·연구대상 위주. 예) 노동시장·젠더정치·복지국가
- 사회과학·사회문제 등 상위분류명 제외.
- 경제경영이면 산업·시장·전략 구체어.
```

### 개편 후 (유형 지정 방식)
```
유형A 분야명         [1개, 필수]  → 교양사회학·비교정치학·미디어사회학
유형B 이론·사상·운동 [0-1개]     → 페미니즘·복지국가론·포퓰리즘
유형C 사회현상·이슈  [0-1개]     → 젠더갈등·양극화·기후정치
유형D 지역·제도      [0-1개]     → 한국정치·유럽복지·남북관계
유형E 트렌드어       [0-1개]     → AI거버넌스·플랫폼자본주의
```

**이미 개편 완료된 분야**: 문학, 철학, 역사, 심리학 *(이전 작업)*

## 3. 오늘 신규 개편: 사회과학 분야

### 3-1. 사회과학 → 경제경영 / 사회정치 분리

기존에 `사회과학`과 `경제경영`이 동일한 프롬프트를 공유하고 있어 키워드 품질이 일관되지 않았음. 알라딘 카테고리 기반으로 두 그룹으로 분리.

| 그룹 | 대상 알라딘 카테고리 | 핵심 키워드 방향 |
|---|---|---|
| **사회정치** | 사회과학 > 사회학, 정치학, 언론, 법학 | 젠더갈등·복지국가·언론자유·AI거버넌스 |
| **경제경영** | 경제경영 > 경영일반, 마케팅, 재무, 창업 | 마케팅전략·린스타트업·ESG경영·재무관리 |

### 3-2. 라우팅 오류 수정

`사회과학 > 교육학` 카테고리 도서가 사회정치 프롬프트를 받고 있던 문제 수정 → **교육 그룹**으로 재라우팅

```
수정 전: 사회과학 > 교육학 → 사회정치 프롬프트
수정 후: 사회과학 > 교육학 → 교육 프롬프트  ✓
```

### 3-3. 차단어 목록 보강

카테고리 fallback 과정에서 상위분류어가 최종 키워드로 삽입되는 현상 차단:

> 추가 차단어: `사회학`, `정치학`, `행정학`, `사회학일반`, `정치학일반`, `교육일반`, `교육학`, `경영일반`, `경제일반`

## 4. 배치 처리 스크립트 개선

기존에 ISBN이 소스코드에 하드코딩되어 있던 구조를 **CSV 파일 입력 방식**으로 변경.

```bash
# 기존 배치 결과 CSV를 입력으로 재처리 (기존 키워드와 비교 가능)
python batch_653.py "653_배치_결과.csv"

# ISBN만 있는 단순 목록
python batch_653.py "isbns.csv"
```

출력 CSV에 **기존키워드 컬럼** 추가 → 프롬프트 변경 전후 결과 엑셀에서 나란히 비교 가능
