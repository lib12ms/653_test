## 1. 문제 인식 (출발점)

기존 프롬프트는 분야별로 AI가 자유롭게 키워드를 추출하도록 설계되어 있었으나, 다음과 같은 저품질 키워드가 반복적으로 생성됨:

- **줄거리 파생 복합어**: `자아희생`, `생존본능`, `지구생존` 등
- **상위분류어 단독 출력**: `사회과학`, `사회학`, `교육학`, `경상계열` 등
- **감상어·메타표현**: `따뜻한`, `감정서사`, `서사구조` 등
- **텍스트 fallback 쓰레기 토큰**: `Part`, `Chapter`, `내가`, `찾기`, 어미(`~는`, `~면`)로 끝나는 조각어

**해결 방향**: 분야별로 키워드 유형(슬롯)을 사전에 지정하고, 그 유형 안에서만 추출하도록 프롬프트를 보수적으로 재설계 — **"유형A-E 슬롯화" 방식**. 공통 구조는 다음과 같음 (분야마다 세부 항목은 다름):

```
유형A 분야명           [1개, 필수]   — 단독 상위분류어 금지, 반드시 수식어/하위개념 결합
유형B 이론·사상·기법   [0-1개]      — 학술 DB·업계에 이미 등재된 용어만, 저자 고유 조어 금지
유형C 현상·대상·개념   [0-1개]      — 설명·목차에 근거 있을 때만
유형D 인물·시대·배경   [0-1개]      — 설명·목차 명시 시만, 추정 금지
유형E 트렌드·응용맥락  [0-1개]      — 설명 명시 시만
```

### 2-1. 06-21 세션 — 사회과학 분야 분리 + 배치 스크립트 개선

|---|---|---|
| 유형A (필수) | 구체 사회과학 분야명 (단독 '사회학'·'정치학' 금지) | 교양사회학·비교정치학·미디어사회학·국제관계학·행정학·언론학 |

금지: 단독 '교육'·'학습', 단독 '기초'·'입문'·'고급', 계열명 단독(경상계열·이공계열)

**문학**: 시집 처리 가이드 유형A-C에 추가 — 시집이면 유형A(현대시·서정시·자연시, 단독 '시' 금지), 유형B(애도·상실·고독·죽음 등 반복 모티프만), 유형C(시집 전체를 관통하는 시대만). 시 분야를 별도 카테고리로 만들 필요는 없다고 결론.

**생활실용**: 유형A 건축 하위 예시 확장 (기존 주거건축·상업건축·인테리어 → 공공건축·박물관건축·종교건축·도시공원·조경설계·건축역사·도시계획 추가)

**필터·버그 수정**

- **LOW_VALUE·차단어 대폭 확장**: `경상계열`, `이공계열`, `Part`, `Chapter`, `내가`, `찾기` 등 + 한국어 어미 필터(`~는`, `~면`) 신규 추가
- **`build_forbidden_set` 과다차단 수정**: 제목 전체연결 문자열 제거로 "토익스피킹" 같은 정상 복합어가 부분매칭으로 차단되던 문제 해소
- **`should_keep_keyword` 과다차단 수정**: "눈물이"가 "눈물"을, "역사소설"이 "역사"를 차단하던 포함관계 체크 제거
- **`category_fallback_used` 오탐 수정**: 양호 키워드 4개가 나와도 fallback 플래그가 True로 잘못 설정되어 점수 페널티를 받던 문제 → 실제 추가 여부 기준으로 변경
- **배치 검토 기준 완화**: `REVIEW_SCORE_THRESHOLD` 0.70→0.55, `REVIEW_MIN_KEYWORDS` 5→4 (4개 양호 키워드 도서가 불필요하게 검토 대상으로 잡히는 현상 해소)

**테스트 결과**: 세션 시작 시점 기준 96권 평균 품질점수 **0.657**, 검토 필요 **41권**

## 4. 설계 검토 — ISBN 부가기호(KDC) 기반 분야 라우팅 (07-03, 미구현)
